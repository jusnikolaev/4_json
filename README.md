# Prettify JSON

Pretty JSON viewer. 

# Quickstart

Example of script launch on Linux, Python 3.5:
<pre>
    <code>
$ python3 pprint_json.py --path "filepath"
    </code>
</pre>

    <code>
{
   "firstName": "",

       ""
   ]
}
    </code>
</pre>
